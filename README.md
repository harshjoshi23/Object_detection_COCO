# Object Detection on COCO

A computer vision project for experimenting with **object detection** workflows on COCO-style data. The project focuses on detecting and localizing objects in images using bounding boxes and class labels.

---

## Project highlights

- Explored object detection as a core computer vision task.
- Worked with COCO-style object categories and annotation concepts.
- Practiced image preprocessing, inference, and result visualization.
- Built foundation for later work in segmentation, visual inspection, and deep learning pipelines.

## Why this project matters

Object detection is used in autonomous driving, robotics, industrial automation, surveillance, retail analytics, and quality inspection. It is a core task that bridges classification and segmentation by identifying both **what** is present and **where** it appears.
